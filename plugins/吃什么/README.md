## 🧩 交互流程
| 你发的              | 机器人回应              | 说明                    |
|--------            |--------              |------                   |
| `/今天吃什么`       | 列出 100+ 道菜并提示删菜 | 进入点餐模式             |
| `/今天吃什么 用法`  | 直接发说明书图片          | /吃什么 用法 同样有效     |
| `0`               | 不删除，立即随机选菜      | 跳过删菜                 |
| `1~N`             | 删除对应序号菜品         | 可多次删，每次都会等待 60 s |
| `不想吃`           | 换一道新菜并重新搜图      | 旧菜永久拉黑              |
| `好耶`            | 结束点餐并祝福            | 随时可发，立即清会话       |
| **60 s 内无操作** | 超时提示已结束            | 需重新发 `/今天吃什么`      |
